# .github

Organization-wide community health files and profile for [Agentic Stacks](https://github.com/agentic-stacks).

## Contents

| File | Purpose |
|------|---------|
| `profile/README.md` | Org landing page shown at [github.com/agentic-stacks](https://github.com/agentic-stacks) |
| `CONTRIBUTING.md` | Default contribution guidelines for all repos |
| `CODE_OF_CONDUCT.md` | Community standards (Contributor Covenant) |
| `FUNDING.yml` | GitHub Sponsors configuration |

## Auto-Updates

The stack catalog table in `profile/README.md` is automatically updated by the [registry sync workflow](https://github.com/agentic-stacks/registry/actions). When stacks are added or removed from the org, the table regenerates between the `<!-- STACKS-TABLE-START/END -->` markers.
